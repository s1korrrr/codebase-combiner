# Release Notes — 0.1.0

Codebase Combiner helps developers turn a selected local codebase into one review-ready text or Markdown document.

This release runs on Apple-silicon Macs (`arm64`) with macOS 13 or later. It does not include an Intel or universal build.

- Combine selected source and text files with configurable filters.
- Preview, copy, and save output locally.
- Recover the last prepared output without exposing its contents until requested.
- Clear recovery data from inside the app.
- Use improved keyboard/accessibility descriptions and direct Privacy/Support links.
- Benefit from bounded scanning, superseded-work protection, and safer output handling.

All processing remains on the Mac; the app has no account, analytics, ads, or network service.
