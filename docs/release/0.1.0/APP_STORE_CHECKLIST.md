> Codebase Combiner is an offline macOS utility for preparing source files for review. Choose a local folder, apply file and extension filters, select the files you need, then build one plain-text or Markdown output. Preview it, copy it, or save it to a location you choose. The app keeps an optional local recovery draft and lets you clear it at any time. It has no account, ads, analytics, or network service.
